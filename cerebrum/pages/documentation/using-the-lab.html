Using the lab

The NERVE lab is one in-browser workspace for evaluation, optional live completions, calibration, and exports. Open the NERVE lab.

Evaluate (no network)

Fill Prompt and Response (and optional Context), choose a label for your records, and click Evaluate. No API key required. Use Run all presets to batch-score built-in test cases (still no API).

Live (calls an LLM API)

Paste a provider API key, pick model and provider, enter Prompt and optional Context, then Live. The app merges prompt + context for the model (see Live API and keys), then runs NERVE on the raw fields and the model reply.

Epsilon (ε)

The slider sets the admissibility threshold for Φ_e adjusted. Keys stay in the browser; nothing is sent to Reverie’s servers for scoring.

Calibration (additive)

Read-only post-hoc analysis on the table or a JSON file: EFCG by label, AUC, ε-sweep, optional lab profile and “apply ε to slider.” It does not re-call the NERVE path or change stored row scores in place—see the in-app copy and Exports and data.
