Exports and data

Exports are generated in the browser. Large bundles use JSZip from a public CDN; if the CDN is blocked, the app falls back to separate downloads where implemented.

JSON + report (zip when available)

Includes results with per-row signals, session kappa, AUC, and a plain-text session report. Re-import the JSON via Import or the calibration Load JSON into table path to restore rows.

CSV (zip when available)

Tabular dump of the results grid for spreadsheets.

Session memory

Recent rows may be stored in localStorage for convenience; clear the session from the app if you need a clean table.
